m1 nvm 安装 node 异常

clang: error: no such file or directory: 'CXX=c++'

Aug 27, 2021
阅读时长: 1 分钟

问题描述

解决方法

需要先执行以下指令再进行安装

arch -x86_64 zsh

应该是因为 v14 及以前的版本不支持 arm 版本的处理器，所以需要使用 Rosetta 2 来进行安装。
